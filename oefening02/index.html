Boeken lezen

Iets wat ik heel graag doe in mijn vrije tijd is boeken lezen. Hierbij kan ik mij ontspannen.
Het geeft mij ook een heel rustgevend gevoel. Elke keer als ik een boek lees zit ik zo in het verhaal
dat ik alles wat er rond mij gebeurd gewoon vergeet. Ik lees verschillende soorten genres.

De soorten genres die ik graag lees

Fantasie: hierbij kan ik zelf mijn fantasie de loop laten gaan.
Romantiek: dit vind ik gewoon leuk om te lezen omdat het vaak ook een wijze les met zich meebrengt.
Historisch: ik vind het heel informatief. Het liefste lees ik over de wereld-oorlogen.

Wanneer is dit begonnen?

Het is allemaal begonnen in de lagere school. Daar hadden wij elk jaar boekopdrachten.
In de loop van de jaren is mijn liefde voor boeken lezen groter geworden.
In het secundair onderwijs begon ik steeds meer boeken te lezen. Maar ook meer genres. Nu lees ik regelmatig nog eens een boek wanneer ik tijd heb.
